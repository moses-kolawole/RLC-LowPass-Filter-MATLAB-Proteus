# RLC Low-Pass Filter Frequency Response Analysis Using MATLAB and Proteus

## Overview

This project presents the numerical and simulation-based analysis of an RLC Low-Pass Filter using MATLAB and Proteus.

The project investigates the frequency response of the filter using:
- MATLAB simulation
- Numerical differentiation methods
- Proteus circuit validation
- MATLAB GUI audio filtering demonstration

The cutoff frequency was estimated using:
- Forward Difference Method
- Backward Difference Method
- Central Difference Method

The project also includes practical validation using Proteus simulation and a MATLAB GUI for real-time audio filtering demonstration.


# Project Objectives

- Analyze the frequency response of an RLC low-pass filter
- Generate Bode magnitude plots
- Apply numerical differentiation methods
- Estimate cutoff frequency
- Validate filter behavior using Proteus
- Demonstrate practical audio filtering using MATLAB GUI

# Results and Conclusion

The project successfully demonstrated the frequency response characteristics of an RLC low-pass filter.

Among the numerical differentiation methods:
- The Central Difference Method provided the most accurate cutoff frequency estimation.
- MATLAB and Proteus results closely matched the analytical calculations.

The project validates both theoretical and practical filter behavior.
